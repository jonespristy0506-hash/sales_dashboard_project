# Sales Performance Dashboard

## Project Overview
This project analyzes sales data using Power BI to identify trends, top-performing products, and regional performance.

## Tools Used
- Power BI
- Python
- Pandas

## Dashboard Features
- Total Sales
- Total Profit
- Total Orders
- Monthly Sales Trend
- Sales by Category
- Top Products by Revenue
- Regional Sales Distribution

## Key Insights
- Technology category generates the highest revenue.
- Sales increase during the holiday season.
- West region contributes the most sales.

## Files in this Repository
- sales_dashboard.pbix – Power BI dashboard
- clean_sales_data.csv – dataset used
- dashboard.png – dashboard screenshot
